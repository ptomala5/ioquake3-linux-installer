# ioquake3 + cpma + extras

## Runing the game
`cd ~/ioquake3 && ./cnq3-x64`

## Troubleshhoting
##### Game appears on the wrong screen
Change or add `seta r_monitor` in `~/.q3a/cpma/q3config.cfg`. Values start from `0`
